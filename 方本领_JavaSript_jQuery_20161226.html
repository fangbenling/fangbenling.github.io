jQuery选择器

	jQuery的基本设计思想和用法 就是“选择某个网页元素，然后对齐进行某种操作”。  使用jQuery的第一步，
	如何选择一个表达式，放进构造函数  “jQuery() 简写:$”通过CSS选择器获取HTML中的元素。以下是
	实现效果 与代码。jQuery书写的正确格式:以标签选择器为例子：
	实例1:
	$(document).ready(function(){
		$('btn').click(function(){
			 $('p').css("background","red")
		});
	});
	实例2:
	 $(document).ready(function(){
			$('.btn').click(clicks)
			function clicks(){
				$('p').css("background","red")
			};
		});
		以上使用选择器 且添加了点击事件 一个btn点击命令  一个p改变CSS样式;
	 选择器有很多: 一、基础选择器：    id  class *(通用)  元素       组合
	               二、层级选择器： 后代$("div span")   子代$("div>span")  相邻$("#one+span")  兄弟$("#one~span")
	               三、过滤选择器： :first--$("div:first")选取所有div>元素中第一个div>元素。
	                       :animated--$("div:animated")选取正在执行动画的div元素 
	               四、属性选择器:  属性与过滤器
	               五、表单域选择器： :input--$(":input")选取所有input>里的标签元素
	                       :text--$(":text")选取所有的单行文本框
	                       :password--$(":password")选取所有密码框
	                       :radio
	                       :chaeckbox
	                       :submit
	                       :image
	                       :reset   重置按钮
	                       :button
	                       :file   上传域
	                       :hidden 不可见元素
