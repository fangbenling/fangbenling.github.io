HTML5 web存储

localStorage 和 sessionStorage

客户端存储的两个对象：

	  1.localStorage 没有时间限制的数据存储
	  2.sessionStorage 针对一个session的数据存储.
	  在使用web存储前 应检查浏览器是否支持  两种存储
	  两者的区别：前者存储的数据不管过多少天 都可以访问获取  后者指针对一个session  当用户关闭浏览器窗口后  数据会被删除
	    input type="" name="" id="age" value="" />
	    input type="" name="" id="sex" value="" />
	    input type="" name="" id="name" value="" />
	    input type="" name="" id="btn" value="" />

			  			try{
			  					window.localStorage.setItem("name"+i,nameT) //后面name是传的参数value值       前面是键值（自己取得名字）

			  	    	window.localStorage.getItem(_item);
			  	    },
			  	    clear:function(){     //删除所有数据 不需要填写键值 和 参数
