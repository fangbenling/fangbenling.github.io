1.面向对象基础

JavaSript一切皆对象 基于对象 比如 我们创建一个var的声明就是创建了一个person对象，我们基于person对象 他有自己的属性 性别 年龄 也有自己的方法 看书 唱歌 都是

 function test(){
 this.x=1;
 console.log(this.x) //结果为1  此处this指代window
	}
test();

this 代表函数运行时，自动生成的一个内部对象，只能在函数内部使用 ,函数使用场合不同，this的值也不同（指代的值也不同） 但是有个总的原则 谁调用函数 他就指代那个对象

四种This指代

一、最基本的函数调用

实例1：
 function test(){
 	this.x = 1;
 	console.log(this.x);
 }
 test();// 1

函数调用时 1赋值给x 

实例2：
 var x = 1;
 function test(){
   console.log(this.x)
 }
test(); //1

二、作为对象方法调用

 function test（）{
console.log(this.x)
 }
 var o = {};   
 o.x = 1;  
 o.m = test;  //把函数给o.m这个对象  让这个对象调用者个函数  此时This 指代这个对象
 o.m(); //1

三、作为构造函数调用

实例1：
 function test(){
 	this.x = 1;
 }
 var str  = new test():
 console.log(str.x)// 1

所谓构造函数，就是通过这个函数生成一个新对象（object）这时 this指代这个新对象

 实例2：为了表明这时this不是全局对象  修改如下
 var x = 2;

 this.x = 1	
 }

四、 apply调用

apply()是函数对象的一个方法，它的作用是改变函数的调用对象， 它的第一个参数就表示改变后的调用这个函数的对象。因此，this指的就是这第一个参数。

 var x = 0;
　　function test(){
　　　　alert(this.x);
　　}
　　var o={};
　　o.x = 1;
　　o.m = test;
　　o.m.apply(); //0

apply()的参数为空时，默认调用全局对象。因此，这时的运行结果为0， 证明this指的是全局对象。

如果把最后一行代码修改为

o.m.apply(o); //1

运行结果就变成了1，证明了这时this代表的是对象o
