动画所学知识点

   script type="text/javascript">
    	$(document).ready(function(){
    		$(".tab li").mouseover(function(){
    			$(this).addClass('active').siblings().removeClass('active')
    			var idx = $(this).index(); //索引选择图片的索引号
    		     console.log(idx)
    			$('.main').eq(idx).addClass('selected').siblings('.main').removeClass('selected')
    		})
    	})
    
    
     通过事件去触发$(".tab li")函数  $(this)指代的是当前class下的li ;
     addClass:增加样式;
     removeClass:移除样式；
     siblings:选择所有兄弟同胞（除去自己）
      $(this).index():对class下的所有li 索引号 （从0开始）
      eq(n):括号里填写 数字  和上面的li建立对应的链接 
      选择不同的li   就会有不同图片与之对应

轮播

          
    用法和上面的使用类似 都是用siblings()同胞   和this   index   索引  与之一一对应，还有animate({style},speed,fun)自定义动画 写法规范
